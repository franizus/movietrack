El objetivo de este proyecto es albergar informacion acerca de series
de tv o peliculas para que los seguidores de las mismas puedan saber
la informacion mas actualizada acerca de las series.

El usuario debe tener una cuenta en el sistema para poder acceder a las funcionaldiades
que tiene el mismo. Es importante brindar información verídica para que el ususario pueda
obtener la cuenta. Una vez que el usuario haya creado la cuenta, podrá acceder a la información de todas
las series y películas que se encuentran registradas en el sistema

Es importante mencionar que este sistema no es:

- Un sitio de descargas de películas o series.
- Una red social.
- Un sitio de comercio eléctrónico de series o películas.
